## About This MicroSim

Compare four timeline layout styles to understand when each is most appropriate.

### Learning Objectives

1. Compare timeline layout styles
2. Select appropriate styles for different content types
3. Understand how layout affects data presentation

## How to Use

1. **Browse Gallery**: View all four styles with the same data
2. **Click to Expand**: See larger examples with descriptions
3. **Use Bottom Buttons**: Quick-switch between styles
4. **Compare**: Note how the same events look in each layout

## Timeline Styles

| Style | Best For |
|-------|----------|
| **Horizontal Linear** | Simple sequences, limited events |
| **Vertical Scroll** | Rich content, long timelines |
| **Centered Alternating** | Dense timelines, presentations |
| **Parallel Tracks** | Comparing multiple sequences |

## Lesson Plan

**Grade Level**: Undergraduate
**Duration**: 15-20 minutes
**Bloom's Level**: Understand (L2)

### Activities

1. **Exploration** (5 min): View each style with the same data
2. **Analysis** (5 min): Identify strengths of each layout
3. **Application** (10 min): Given historical data, choose appropriate style

### Assessment

Students demonstrate mastery by:

- Explaining why vertical works for rich content
- Choosing parallel tracks for comparing sequences
- Identifying when horizontal is insufficient
