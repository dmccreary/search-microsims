# Creative Commons License Chooser

An interactive tool to help you select the appropriate Creative Commons license for your MicroSims by answering simple questions about commercial use and modifications.

## How to Use

1. **Answer the questions** in the left panel:
   - **Allow commercial use?** - Can others profit from work based on yours?
   - **Allow modifications?** - Can others adapt or remix your work?

2. **Watch the decision path** highlight in the center flowchart

3. **View your recommendation** in the right panel with:
   - License badge with icons
   - Plain English explanation
   - Ready-to-copy JSON for your metadata

### Quick Option

Click **"Maximum Freedom (CC0)"** to waive all rights and dedicate your work to the public domain.

### Note on Attribution

Attribution (BY) is actually a third dimension of Creative Commons licensing. All CC licenses except CC0 require attributionâ€”meaning others must credit you when using your work. In this tool:

- **CC0** = No attribution required (public domain)
- **All other licenses** = Attribution required (BY is implicit)

To keep the interface simple, we assume most creators want attribution. The "Maximum Freedom (CC0)" button is provided for those who want to waive all rights, including attribution.

## License Types

| License | Commercial Use | Modifications | Notes |
|---------|---------------|---------------|-------|
| CC0 | Yes | Yes | Public domain - no restrictions |
| CC BY | Yes | Yes | Requires attribution |
| CC BY-SA | Yes | Yes (ShareAlike) | Derivatives must use same license |
| CC BY-ND | Yes | No | Commercial OK, but no modifications |
| CC BY-NC | No | Yes | Non-commercial only |
| CC BY-NC-SA | No | Yes (ShareAlike) | Non-commercial + same license |
| CC BY-NC-ND | No | No | Most restrictive - sharing only |

## About This MicroSim

This tool was created to help MicroSim authors understand and apply Creative Commons licenses to their work. It implements the decision logic used by Creative Commons to guide license selection.

**Learning Objective:** Students will recommend appropriate Creative Commons licenses for different MicroSim sharing scenarios by analyzing use case requirements against license permissions.

**Bloom Level:** Evaluate (L5) - Recommend
