## About This MicroSim

This Analyze-level (L4) MicroSim has students classify educational activities by their cognitive level.

### Learning Objectives

1. Classify MicroSim activities by Bloom's Taxonomy level
2. Distinguish between lower-order and higher-order activities
3. Match verbs to appropriate cognitive levels
