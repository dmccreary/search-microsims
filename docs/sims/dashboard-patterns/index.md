## About This MicroSim

Construct effective dashboard layouts by dragging components onto a grid canvas.

### Learning Objectives

1. Construct effective dashboard layouts
2. Select appropriate components for different purposes
3. Apply dashboard design principles

## How to Use

1. **Drag Components**: Drag from library to canvas grid
2. **Apply Templates**: Click a template to auto-populate
3. **Remove Components**: Click the Ã— on any placed component
4. **Clear All**: Reset the canvas to start over

## Dashboard Components

| Component | Purpose |
|-----------|---------|
| **KPI Card** | Key metrics at a glance |
| **Line Chart** | Trends over time |
| **Bar Chart** | Category comparisons |
| **Pie Chart** | Part-to-whole relationships |
| **Data Table** | Detailed data access |
| **Status** | Current state indicators |

## Lesson Plan

**Grade Level**: Undergraduate
**Duration**: 20-25 minutes
**Bloom's Level**: Apply (L3)

### Activities

1. **Exploration** (5 min): Try the three templates
2. **Construction** (10 min): Build a custom dashboard for a scenario
3. **Critique** (5 min): Compare designs with classmates
4. **Discussion** (5 min): What makes a dashboard effective?

### Assessment

Students demonstrate mastery by:

- Creating a balanced dashboard layout
- Choosing appropriate components for a given purpose
- Explaining their design decisions
