# Chapters

This textbook is organized into 14 chapters covering 200 concepts.

## Chapter Overview

1. [Introduction to MicroSims](01-intro-to-microsims/index.md) - Introduces foundational concepts including purpose, structure, interactivity, and reusability.
2. [MicroSim File Organization](02-microsim-file-organization/index.md) - Covers directory structure and essential files that comprise a MicroSim package.
3. [Metadata Fundamentals](03-metadata-fundamentals/index.md) - Introduces metadata concepts, Dublin Core standard, taxonomies, and classification systems.
4. [Dublin Core Elements](04-dublin-core-elements/index.md) - Provides detailed coverage of all 15 Dublin Core metadata elements.
5. [JSON and Data Structures](05-json-and-data-structures/index.md) - Covers JSON syntax, schema validation, and data quality metrics.
6. [Search Fundamentals](06-search-fundamentals/index.md) - Introduces information retrieval, precision, recall, keyword and boolean search.
7. [Faceted Search and Client-Side Search](07-faceted-search-client-side/index.md) - Covers faceted search implementation, filter controls, and ItemsJS library.
8. [Embeddings and Semantic Search](08-embeddings-semantic-search/index.md) - Introduces vector embeddings, cosine similarity, and dimensionality reduction techniques.
9. [Data Pipelines and Aggregation](09-data-pipelines-aggregation/index.md) - Covers web crawling, GitHub API, repository mining, and data aggregation.
10. [Educational Foundations](10-educational-foundations/index.md) - Covers grade levels, Bloom's Taxonomy, curriculum standards, and assessment.
11. [Learning Theory and Pedagogy](11-learning-theory-pedagogy/index.md) - Explores cognitive load theory, scaffolding techniques, and learning theories.
12. [Visualization Types](12-visualization-types/index.md) - Covers visualization types including animations, charts, diagrams, and interactive displays.
13. [User Interface and Controls](13-user-interface-controls/index.md) - Covers interaction levels, control widgets, and layout patterns.
14. [Technical Implementation](14-technical-implementation/index.md) - Covers JavaScript libraries, responsive design, accessibility, and analytics.

## How to Use This Textbook

Chapters are organized to respect concept dependencies - each chapter builds on knowledge from previous chapters. Foundational concepts appear in early chapters, with more advanced topics in later chapters. While you can jump to specific topics of interest, completing chapters in order ensures you have the prerequisite knowledge for each new concept.

---

**Note:** Each chapter includes a list of concepts covered. Make sure to complete prerequisites before moving to advanced chapters.
